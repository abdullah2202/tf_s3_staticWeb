Cloud-Native Static Website Deployment

Terraform, S3 & CloudFront CI/CD Pipeline

This project showcases a professional, fully automated pipeline for hosting a static website on Amazon Web Services (AWS). It leverages Infrastructure as Code (IaC) principles to provision and manage all cloud resources.

  • Infrastructure as Code (IaC): Terraform manages all AWS resources.
  • CloudFront CDN: Provides a fast, global delivery network with automatic HTTPS.
  • S3 Security: Content stored privately, only accessible via Origin Access Control (OAC).
  • GitHub Actions: CI/CD pipeline for automatic infrastructure changes, content sync, and cache invalidation.

Any code changes pushed to the repository automatically trigger the pipeline, ensuring immediate deployment and consistency across the environment.
